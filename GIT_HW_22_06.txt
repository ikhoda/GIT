 1. 	q/ Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
	a/ С пмощью команды "git clone <ssh ссылка на репо с гитхаба>"клонируем себе в локальную папку содержимое репозитория на Гитхабе
	предварительно необходимо в настройках профиля ГХ привязать компьютер по SSH протоколу.
	Затем с помощью команды git pull качаем содержимое репозитория в локальную папку.

2. 	q/ Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
	a/ заходим в произвольную папку с файлами
	инициируем локальный репозиторий. git init
	git add <file_name> добавляем файлы из папки к коммиту.
	git commit -m "" закомитить.
	Чтобы отправить на гитхаб, привязываем папку на компе к созданному репозиторию на ГХ (см. пункт 1) git clone
	Заливаем на Гитхаб: git push

3.	q/ Как получить на локальный компьютер изменения с github.com
	a/ С помощью команды git pull загружаем файлы из ГХ. При этом видно какие файлы были изменены.

4.	Какую информацию нам дают команда :
		git status- статус файлов (добавлен, готов к комиту, закомичен, не закомичен)
		git log- лог комитов в репозитории на ГХ
		git remote -v варианты что мы можем сделать с репозиторием (загрузить/выгрузить файлы)
